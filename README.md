# ⚙️ Agentic OS - 31 Harbor Agency Dashboard

> An AI-powered workflow management and automation hub, inspired by the Agentic OS concept, designed to orchestrate tasks and provide observability across various business domains for 31 Harbor, Party Favor Photo, and XMRT DAO.

## Overview

This dashboard has been significantly upgraded to provide a dedicated **Agentic OS** interface for each client company (31 Harbor, Party Favor Photo, and XMRT DAO). Instead of a single, unified view, each company now has its own comprehensive suite of tools and metrics, mirroring the full Agentic OS architecture described in the video.

For each client, the dashboard now features:

-   **The Conductor**: A conceptual central AI orchestrator for that specific company.
-   **Domain-Specific Workflows**: Organized into Research, Content, Community, and Agency domains, each with specific "Skill Cards" tailored to the company's operations.
-   **Automation Layer**: Visualizes the integration of various tools and software assets (GitHub, Slack, Email, Obsidian, Resend, Cloudflare, Analytics, Auth) relevant to the company.
-   **Memory Vault**: A structured representation of an Obsidian-based knowledge base for the company, categorized into Raw data, Wiki (processed reports), and Output (deliverables).
-   **Observability Dashboard**: Provides real-time metrics on agentic activity, API usage, active routines, and success rates specific to the company, along with quick skill execution buttons.
-   **Recent Email Activity**: A dedicated table displaying detailed information about recent emails, including timestamp, sender, recipient, subject, and status.
-   **General Activity Log**: Displays recent system activities and other events pertinent to the individual company.

This per-company organization allows for a more focused and tailored management of AI-driven workflows for each client.

## Features

-   **Per-Company Agentic OS**: Dedicated sections for each client, providing a complete Agentic OS experience.
-   **Detailed Email Activity**: Real-time display of email sender, recipient, subject, and status for each company.
-   **Modular Design**: Easily expandable with new domains and skill cards for each company.
-   **Real-time Data Integration**: Fetches and displays live data from `data.json` (generated by `scripts/update_stats.py`), dynamically populating company-specific metrics.
-   **Responsive UI**: Optimized for various screen sizes.
-   **Dark Theme**: Modern and professional aesthetic.

## Structure

-   `index.html`: The main dashboard interface, dynamically generating Agentic OS sections for each company.
-   `data.json`: Stores dynamic data for the dashboard, updated by `scripts/update_stats.py`.
-   `scripts/update_stats.py`: Python script responsible for fetching data (e.g., from Resend API) and updating `data.json`.
-   `CNAME`: Custom domain configuration for GitHub Pages.
-   `.github/workflows/deploy.yml`: GitHub Actions workflow for deployment to GitHub Pages.

## Setup & Deployment

This dashboard is deployed via GitHub Pages. The `deploy.yml` workflow handles automatic deployment on pushes to the `main` branch. The custom domain `agency.31harbor.com` is configured via the `CNAME` file.

To update the dynamic data, ensure `scripts/update_stats.py` is run periodically to refresh `data.json`.
